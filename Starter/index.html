Summary: Latitude vs. X

The purpose of this project was to analyze weather in relationship to distance to the equator. In order to complete this analysis, we pulled weather data from the OpenWeather API for 500 different cities with differing longitudes and latitudes.

Once the dataset was compiled, we used Matplotlib to plot latitude vs. temperature, wind speed, humidity, and cloudiness. All relevant visualizations and analysis compiled during this study can be found throughout this site.
